Agachamento Livre

Descrição

O agachamento livre é um exercício completo para o fortalecimento das pernas, glúteos e core.

Músculos trabalhados

  • Quadríceps
  • Glúteos
  • Core

Passo a passo

  1. Afaste os pés na largura dos ombros.
  2. Mantenha a coluna reta.
  3. Flexione os joelhos, levando o quadril para trás.
  4. Desça até onde for confortável.
  5. Suba empurrando o chão pelos calcanhares.

Prescrição

Séries: 3

Repetições: 12

Descanso: 30s
